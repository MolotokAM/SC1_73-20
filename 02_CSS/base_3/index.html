Блок навигации
  • Ширина 15em
  • Должен присутствовать цвет фона
  • Распологаться слева
  • Блок навигации наполните контентом достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере.

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствать цвет фона
Во всех "статьях" сделайте отступ от соседей равным .2em, при помощи свойства margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех "статьях" сделайте отступ снизу от соседеё равным .2em, при помощи свойства margin-bottom.
Картинка вставлена при помощи тега, плавание организованно по правому краю внутри данной статьи для этой картинки.
Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещённым внутрь контентом.
